Currency Settings

Select your preferred currency. This will be used throughout the application for all monetary values.

Changing the currency will update all displayed amounts throughout the application.

Import Files

Import month data from JSON, CSV, or HTML files. You can select a folder (modern browsers) or individual files (all browsers).

Modern browsers: Click to select the data/months/ folder directly.
All browsers: If folder selection isn't available, you'll be prompted to select individual files.

Select one or multiple files from the data/months/ folder (JSON/CSV) or reference/ folder (HTML). The format will be detected automatically.

Required for CSV and HTML files. JSON files include year information in the data.

Export & Delete Months

Export months in various formats (JSON, CSV, or HTML) or delete months.

Export Selected Month: Export the selected month in the chosen format (JSON, CSV, or HTML).
Export All Months: Export all months in the chosen format (JSON, CSV, or HTML).
Delete Selected Month: Permanently delete the selected month. This action cannot be undone.
Note: HTML export creates formatted reports.
